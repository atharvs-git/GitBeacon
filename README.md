# GitBeacon 🔍

A GitHub profile analytics tool that provides deep insights into any developer's GitHub activity.

## 🌐 Live Demo
[gitbeacon.vercel.app](https://gitbeacon.vercel.app)

## ✨ Features

- **Developer Health Score** — Comprehensive analysis with radar chart visualization across 5 dimensions
- **Learning Velocity & Repository Diversity** — Track how fast you're adopting new technologies
- **Developer Comparison** — Compare any two GitHub profiles side by side with overlapping radar charts
- **Language Analysis & Contribution Insights** — Pie chart, bar chart, heatmap, and proficiency scores

## 🛠️ Built With
- React.js
- Vite
- GitHub REST API
- Custom scoring algorithms
- Canvas-based data visualizations

## 🚀 Getting Started
```bash
git clone https://github.com/atharvgupta-dev/GitBeacon.git
cd GitBeacon
npm install
npm run dev
```
